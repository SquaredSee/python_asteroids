# Python Asteroids

A python implementation of asteroids written in pygame

## Controls
* Left Arrow: Rotate left
* Right Arrow: Rotate Right
* Up Arrow: Move forward
* Z: Shoot

## Installation
### For [pipenv](https://pipenv.readthedocs.io/):
```bash
pipenv install
pipenv run python asteroids.py
```
### Manual installation:
```bash
pip install pygame
python asteroids.py
```
